1 Introduction

Pandas (Python Data Analysis Library) is an open-source library designed for data manipulation and analysis. The name is derived from the econometrics term "panel data" — multi-dimensional structured datasets.

FeatureDetail
Built onNumPy
Primary structuresSeries (1D) and DataFrame (2D)
File formatsCSV, Excel, JSON, SQL, Parquet, HDF5, Feather, HTML, and more
IntegrationNumPy, Matplotlib, scikit-learn, SciPy
Missing values.dropna(), .fillna()
Time seriespd.to_datetime(), resample(), rolling()

4 DataFrame

A DataFrame is a two-dimensional, size-mutable, and potentially heterogeneous tabular data structure with labeled axes (rows and columns).

4.1 Creating a DataFrame

import pandas as pd
import numpy as np

# From a dictionary
data = {
    'Name'  : ['Alice', 'Bob', 'Carol'],
    'Age'   : [28, 34, 25],
    'Score' : [85.5, 92.0, 78.3]
}
df = pd.DataFrame(data)

# From a list of dicts
df2 = pd.DataFrame([{'x': 1, 'y': 10}, {'x': 2, 'y': 20}])

# From a NumPy array
df3 = pd.DataFrame(np.random.randn(4, 3), columns=['A', 'B', 'C'])

# From CSV (most common)
df4 = pd.read_csv('data.csv')

# From a list of lists with explicit columns
df5 = pd.DataFrame([[1, 'a'], [2, 'b']], columns=['num', 'letter'])

4.2 DataFrame Index

# Default RangeIndex
print(df.index)              # RangeIndex(start=0, stop=3, step=1)

# Set a custom index
df = df.set_index('Name')

# Reset to default integer index
df = df.reset_index()

# Create with custom index
df6 = pd.DataFrame({'val': [10, 20, 30]}, index=['a', 'b', 'c'])

4.3 Accessing a DataFrame

df['Age']              # single column → Series
df[['Name', 'Age']]    # multiple columns → DataFrame
df.loc[0]              # row by label
df.iloc[0]             # row by integer position
df.at[0, 'Age']        # fast label scalar
df.iat[0, 1]            # fast integer scalar

4.4 Indexing and Selecting Data

# Boolean indexing
high_scores = df[df['Score'] > 80]

# .loc — label-based (inclusive)
subset = df.loc[0:2, 'Name':'Score']

# .iloc — integer-position-based (exclusive end)
subset = df.iloc[0:3, 0:2]

# query() — SQL-like string expressions
result = df.query("Age > 25 and Score > 80")

# where() — replace values not matching condition
df_masked = df.where(df['Score'] > 80)

4.5 Filtering with Multiple Conditions

# Using & (AND), | (OR), ~ (NOT)
result = df[(df['Age'] > 25) & (df['Score'] > 80)]
result = df[(df['Age'] < 30) | (df['Score'] > 90)]
result = df[~(df['Name'] == 'Bob')]

# Using .query()
result = df.query("Age > 25 & Score > 80")

4.6 Merging, Joining, and Concatenating

# merge — SQL-style join
df_merged = pd.merge(df_left, df_right, on='key', how='inner')
# how: 'inner', 'left', 'right', 'outer'

# concat — stack DataFrames
df_combined = pd.concat([df1, df2], axis=0, ignore_index=True)
df_wide     = pd.concat([df1, df2], axis=1)

# join — join on index
df_joined = df1.join(df2, how='left')

# combine_first — fill missing values from another DataFrame
df_filled = df1.combine_first(df2)

4.7 Sorting a DataFrame

# Sort by one column
df.sort_values('Score', ascending=False)

# Sort by multiple columns
df.sort_values(['Dept', 'Score'], ascending=[True, False])

# Sort by index
df.sort_index()

# Rank
df['rank'] = df['Score'].rank(method='dense', ascending=False)

4.8 Pivot Table

pivot = df.pivot_table(
    values     = 'Score',
    index      = 'Dept',
    columns    = 'Gender',
    aggfunc    = 'mean',
    fill_value = 0,
    margins    = True   # adds row/col totals
)

5 Series

A Series is a one-dimensional labeled array capable of holding any data type. It's similar to a single column in a spreadsheet.

5.1 Creating a Series

# From a list
s1 = pd.Series([10, 20, 30, 40])

# With custom index
s2 = pd.Series([10, 20, 30], index=['a', 'b', 'c'])

# From a NumPy array
s3 = pd.Series(np.array([1.1, 2.2, 3.3]))

# From a dictionary (keys become index)
s4 = pd.Series({'x': 100, 'y': 200, 'z': 300})

# Scalar broadcast
s5 = pd.Series(42, index=['a', 'b', 'c'])

5.2 Accessing Elements

s = pd.Series([10, 20, 30, 40, 50], index=['a','b','c','d','e'])

s['a']         # by label → 10
s[['a','c']]   # multiple labels
s.iloc[0]       # by position → 10
s.iloc[1:4]     # slice
s[s > 20]      # boolean → 30, 40, 50

5.3 Binary Operations (auto index alignment)

s1 = pd.Series([1, 2, 3], index=['a','b','c'])
s2 = pd.Series([10, 20, 30], index=['b','c','d'])

print(s1 + s2)   # a=NaN, b=12, c=23, d=NaN
s1.add(s2, fill_value=0)  # fill missing with 0 before adding

5.4 Key Attributes

s.index    # RangeIndex(start=0, stop=5, step=1)
s.values   # numpy array of values
s.dtype    # int64
s.name     # series name
s.shape    # (5,)
s.size     # 5
s.ndim     # 1

5.5 Useful Operations

s = pd.Series([3, 1, 4, 1, 5, 9, 2, 6, 5])

s.unique()                       # unique values
s.nunique()                      # count of unique values
s.value_counts()                 # frequency count
s.nlargest(3)                    # top 3
s.nsmallest(3)                   # bottom 3
s.clip(lower=2, upper=6)         # clip outliers
s.map({1:'one', 2:'two'})          # map values

7 Data Cleaning

7.1 Handling Missing Data

df.isnull()                           # boolean mask
df.isnull().sum()                      # count per column
df.notnull()

# Drop
df.dropna()                            # rows with any NaN
df.dropna(axis=1)                      # columns with any NaN
df.dropna(how='all')                   # only entirely-NaN rows
df.dropna(thresh=3)                    # keep rows with ≥3 non-NaN
df.dropna(subset=['name', 'age'])      # based on specific columns

# Fill
df.fillna(0)                            # scalar
df.fillna(df.mean())                    # column mean
df.fillna(method='ffill')              # forward fill
df.fillna(method='bfill')              # backward fill
df.fillna({'col1': 0, 'col2': 'N/A'}) # per-column

# Interpolate
df.interpolate(method='linear')
df.interpolate(method='spline', order=2)

7.2 Removing Duplicates

df.duplicated().sum()                 # count duplicates
df.drop_duplicates()
df.drop_duplicates(subset=['name'])
df.drop_duplicates(keep='last')       # 'first'(default),'last',False

7.3 Changing Data Types

df['age']   = df['age'].astype('int32')
df['score'] = df['score'].astype('float32')
df['dept']  = df['dept'].astype('category')

# to_numeric with error handling
df['col'] = pd.to_numeric(df['col'], errors='coerce')  # bad → NaN
df['col'] = pd.to_numeric(df['col'], downcast='integer')

# to_datetime
df['date'] = pd.to_datetime(df['date'], format='%d-%m-%Y')

# Smart inference
df = df.convert_dtypes()

7.4 String Manipulations

s = df['name']

# Case

# Search

# Replace

8.4 GroupBy and Aggregation

grouped = df.groupby('dept')

grouped['salary'].agg(['mean', 'min', 'max', 'count'])

    avg_salary = ('salary', 'mean'),
    max_salary = ('salary', 'max'),
    headcount  = ('name',   'count')
)

# transform — broadcasts back to original shape
df['dept_avg'] = df.groupby('dept')['salary'].transform('mean')

# filter — remove groups not meeting a condition
df.groupby('dept').filter(lambda g: g['salary'].mean() > 70000)

8.5 Types of Joins

pd.merge(df1, df2, on='key', how='inner')   # matching keys only

pd.merge(df1, df2, on='key', how='outer')   # all keys from both
pd.merge(df1, df2, left_on='id', right_on='user_id')
pd.merge(df1, df2, on='key', how='outer', indicator=True)
